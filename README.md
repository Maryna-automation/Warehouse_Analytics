## About

This project automates warehouse data analytics using Python, n8n, Google Gemini AI, Notion, and Gmail.  
It processes Excel files, calculates total revenue, taxes, quantities, generates charts, and sends automated reports to Gmail & Notion.

---

## Technologies

- Python (pandas, matplotlib, xlsxwriter, requests, dotenv)  
- n8n Workflow Automation  
- Google Gemini AI (PaLM)  
- Notion API  
- Gmail API  
- Excel files for input/output
